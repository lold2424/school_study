### 정렬 알고리즘의 평가 기준

- 비교 횟수
- 이동 횟수단순핳고 비효율적인것

| 단순하고 비효율적인것 | 복잡하고 효율적인것 |
| --- | --- |
| 삽입 | 퀵 |
| 선택 | 힙 |
| 버블 | 합병 |
|  | 기수 |

**파란색은 안정석이 필수적으로 요구됨**
